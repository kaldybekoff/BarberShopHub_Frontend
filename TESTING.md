## Порядок тестирования

### 1. Auth флоу

**Регистрация:**
1. Открыть `/` — WelcomePage
2. Нажать "Зарегистрироваться" → `/register`
3. Ввести любые данные → отправить
4. Попасть на `/verify` → ввести код `1234`
5. Убедиться что редирект произошёл на `/home`

**Логин:**
1. Открыть `/login`
2. Ввести `user@test.com` / `123456`
3. Убедиться что редирект на `/home`

**Защита роутов:**
1. Открыть `/home` без входа → должен редиректить на `/login`
2. Войти как User, вручную открыть `/admin/dashboard` → редирект
3. Войти как User, вручную открыть `/barbershop/dashboard` → редирект

---

### 2. User флоу

Войти: `user@test.com` / `123456`

**Главная и поиск:**
1. `/home` — карточки барбершопов отображаются
2. `/search` — поиск работает, фильтрация по названию/адресу
3. Кликнуть на карточку → `/shops/1` — детали барбершопа

**Бронирование (главный флоу):**
1. На странице барбершопа нажать "Записаться" → `/booking/1`
2. Шаг 1: выбрать услугу → кнопка "Далее" активируется
3. Шаг 2: выбрать мастера (или "Любой") → выбрать дату → выбрать время → "Далее"
4. Шаг 3: проверить сводку → нажать "Подтвердить запись"
5. Попасть на `/booking-success` — данные записи отображаются
6. Нажать "Мои записи" → `/appointments`

**Мои записи:**
1. `/appointments` — таб "Предстоящие" — карточки записей
2. Переключить таб "Прошедшие" — другой список
3. Нажать "Отменить" на записи — статус меняется

**Профиль:**
1. `/profile` — имя, email, статистика
2. Нажать "Мои записи" → переход на `/appointments`
3. Нажать "Выйти" → редирект на `/`

---

### 3. Barbershop флоу

Войти: `shop@test.com` / `123456`

1. `/barbershop/dashboard` — статистика (4 карточки), график выручки, ближайшие записи
2. `/barbershop/bookings` — список всех записей
3. `/barbershop/services` — список услуг
4. `/barbershop/schedule` — расписание
5. `/barbershop/analytics` — аналитика
6. Sidebar: активный пункт подсвечен, переходы работают
7. Кнопка "Выйти" в sidebar → редирект на `/`

---

### 4. Admin флоу

Войти: `admin@test.com` / `123456`

1. `/admin/dashboard` — статистика (4 карточки), таблицы
2. `/admin/users` — таблица пользователей со статусами
3. `/admin/barbershops` — таблица барбершопов с рейтингами
4. `/admin/reviews` — модерация отзывов
5. Sidebar: навигация между разделами
6. Кнопка "Выйти" → редирект на `/`

## Что проверить дополнительно

| Что | Как проверить |
|---|---|
| localStorage | DevTools → Application → Local Storage → видеть `access_token` и `user` |
| Очистка токена | После выхода `access_token` и `user` удалены |
| Неизвестный маршрут | Открыть `/blabla` → редирект на `/` |
| Шаги бронирования | Нажать "← Назад" на шаге 2 → вернуться на шаг 1 |
| Кнопка "Далее" | Не активна пока не выбран вариант |
